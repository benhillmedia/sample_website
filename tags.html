Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Document Tags

Tag Name Purpose
html HTML specifies that the document is html
head head contains metadata about the HTML document
body defines the main content of the HTML document
title title defines the title of the HTML document

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header abstract semantic tag used to label a part of the page
div division sets the page divisions apart
ol ordered list creates an ordered list- e.g. numbered
ul unordered list creates an unordered list- e.g. bullet points
blockquote blockquote creates a long ("block") quotation

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="https://learnenough.com/">Learn Enough</a> LearnEnough.com
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code show code in plain text <code>content will look like this<code> content will look like this
span span used for grouping and applying styles to inline elements <span>code will look like this<span> code will look like this
li list item defines the items in a list <li>the tag will look like this</li>
